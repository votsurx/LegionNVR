# 🛡️ Legion NVR V4.0 — AI Release

**Legion NVR** — это мощная система видеонаблюдения с искусственным интеллектом на базе YOLOv8.  
Работает на CPU, детектит людей и машины, пишет видео с предзаписью и отфильтровывает ложные тревоги.

---

## 🆕 Что нового в V4.0

- 🤖 **AI-детекция (YOLOv8n)** — гибридный режим MOG2 + нейросеть
- 🎯 **Фильтрация ложных тревог** — минус 80-90% ложных срабатываний
- 🧠 **Умные зоны детекции** — exclude/include с рисованием на кадре
- 📼 **Идеальная запись** — HLS-сегменты 1 сек, склейка без дырок
- ⚙️ **AI-настройки в UI** — выбор классов, уверенность, пропуск кадров
- 🔄 **Живое обновление детектора** — MQTT-команды на лету
- 🛡️ **Стабильность** — детектор не падает, БД не течёт
